## 🔴 修复：打包后 API 无法启动

### 根因
Vite 的 `external` 配置将 `@imput/version-info` (pnpm workspace 包) 标记为 external。electron-builder 打包 asar 时，workspace 的符号链接无法正确跟随，导致运行时 `import "@imput/version-info"` 失败 → API 无法启动 → 所有下载失败。

### 修复
6. **packages/desktop/vite.config.ts** — 1处改动
   - external 函数添加例外：`@imput/version-info` 标记为 bundled（打包进构建输出）
   - 验证：config chunk 从 20.80kB → 23.18kB（多出的 2.38kB 即 version-info 包）
