Why Cobalt?

A native desktop experience built specifically to make downloading public files friendly and fast.

Local Cobalt Core

Embedded with the full Cobalt API engine running directly inside the main process, eliminating reliance on unstable public web instances.

Parallel Downloads

Queue multiple URLs at once. The integrated scheduler will handle parallel downloads dynamically based on your custom slot limit settings.

Clipboard Monitoring

Optionally monitors your clipboard to auto-detect media URLs, showing a subtle interactive prompt to download them with one click.

Smart Drag & Drop

Drag links from your web browser address bar directly and drop them anywhere into the application to initiate downloads instantly.

Zero Tracking or Ads

True to Cobalt's core philosophy—absolutely no telemetry, no tracking scripts, no ads, and no paywalls. Clean and direct saving.

Multi-Language Support

Designed with Svelte 5 and an integrated i18n system supporting multiple locales, automatically synced with your macOS system settings.

Installation Guide

Setting up Cobalt on macOS is quick and straightforward.

1 Download

Get the latest .dmg installer from the Releases page of the repository.

2 Install

Double-click the downloaded .dmg file, then drag the Cobalt icon into your macOS Applications folder.

3 Open

Launch Cobalt from your Applications or Launchpad, and start downloading your favorite content.

Bypassing macOS Gatekeeper (Unverified Developer)

Since the app is built and packaged without Apple Developer ID signing, macOS might show a warning dialog. You can bypass this easily by right-clicking (Control-clicking) Cobalt.app in Finder, selecting Open, and confirming. Alternatively, run this command in Terminal:

xattr -cr /Applications/Cobalt.app
